7. Ascorbic acid, or vitamin C (C6H8O6), is an essential vitamin. It cannot be stored by the body and must be present in the diet. What is the molar mass of ascorbic acid? Vitamin C tablets are taken as a dietary supplement. Ifa typical tablet contains 500.0 mg vitamin C, what amount (moles) and what number of molecules of vitamin C does it contain?


Answer:

The molar mass of vitamin C is 176.12 g/mol.

The number of moles in 500.0 mg vitamin C is 2.84x10^-3 mol

The number of moles in 500.0 mg vitamin C is 1.71x10^21 molecules

Explanation:

Determine the molar mass

The chemical formula for vitamin C is C6H8O6

The molar mass of C6H8O6: The ­atomic mass of C is 12.01 g/mol , H is 1.008 g/mol and O is 16.00 g/mol. We multiply each atomic mass by the number present in the chemical formula and then add them together to get the molar mass. The molar mass of C6H8O6 then is:

C6H8O6        6 C × 12.01 g/mol C  = 72.06 g/mol      

                       8 H × 1.008 g/mol H  = 8.064 g/mol        

                      6 O × 16.00 g/mol O = 96.00 g/mol      

                                                 Sum = 176.124 g/mol

Calculate the number of moles

Given:

mass = 500.0 mg

molar mass = 176.12 g/mol

Solve for number of moles

500 mg * 10^-3 g / 1 mg * 1 mol /  176.12 g = 2.84x10^-3 mol

Calculate the number of molecules

Given:

moles = 2.84x10-3 mol

Avogadro's number = 6.022x1023 molecules

Solve for number of molecules

2.84x10^-3 mol  (6.022x10^23 molecules/ 1 mol) = 1.71x10^21 molecules

13. Vitamin C (ascorbic acid) contains 40.92% C, 4.58% H, and 54.50% O by mass. What is the empirical formula of ascorbic acid?​


Answer:

empirical formula = C₃H₄O₃

Explanation:

Assuming that the mass of the compound is 100 g

mass C = 40.92 g

mass H = 4.58 g

mass O = 54.50 g

Calculating the number of moles of each element

n C = 40.92 g × (1 mol / 12.0107 g) = 3.40696 mol

n H = 4.58 g × (1 mol / 1.00794 g) = 4.54392 mol

n O = 54.50 g × (1 mol / 15.9994 g) = 3.40638 mol

Determining the empirical formula

Representing the empirical formula

empirical formula = CxHyOz

Dividing the number of moles of each element by the least number of moles

x = 3.40696 mol / 3.40638 mol = 1 × 3 = 3

y = 4.54392 mol / 3.40638 mol = 1.33 × 3 = 4

z = 3.40638 mol / 3.40638 mol = 1 × 3 = 3

Writing the empirical formula

empirical formula = C₃H₄O₃
