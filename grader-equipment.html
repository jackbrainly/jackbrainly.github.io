4. Garland High School has grades 9, 10, 11, and 12. The total enrollment is 3600 students. If 40% of the enrollment are 9th graders, 25% are 10th graders, and the 11th graders and 12th graders have equal numbers, then how many students are 12th graders?​


Answer:

630 students

Step-by-step explanation:

G9+G10+G11+G12=3,600, where;

G9 = 40% x 3,600

G10 = 25% x 3,600

G11+G12 = [100% - (40%+25%)] x 3,600, and;

G12 = (G11+G12)/2

Computation:

G9 = 40% x 3,600 = 1,440

G10 = 25% x 3,600 = 900

G11+G12 = [100% - (40%+25%)] x 3,600

              = [100% - (65%)] x 3,600

              = (35%) x 3,600

              = 1,260

G12 = (G11+G12)/2

      = 1,260/2

G12 = 630

Checking:

G9+G10+G11+G12=3,600

1,440+900+630+630=3,600

3,600 = 3,600
