17. Ms. Santos asked renz to draw all the diagonals of a certian polygon on the blackboard. Renz was able to 27 diagonal which His teacher declared correct. What was the given polygon?


Number of diagonals = [n(n-3)]/2     where n = number of sides of the polygon

Find the number of sides, n:

27 = [(n (n-3)]/2

27 = (n² - 3n) / 2

n² - 3n = 27 (2)

n² - 3n - 54 = 0

Factor the quadratic equation:

(n - 9) (n + 6) = 0

n - 9 = 0            n + 6 = 0
n = 9                 n = - 6

Choose the positive n:

n = 9
Number of sides = 9
Polygon with 9 sides = Nonagon

ANSWER:  The polygon was nonagon.
