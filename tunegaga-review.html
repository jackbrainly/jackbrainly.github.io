2. how important is book reviews and literature riviews?​


Answer:

The publication of book reviews gives readers an idea of which books will be valuable in the classroom, in their own scholarship, or just for personal edification. If the reviews are favorable, they can encourage readership. Negative reviews can drive readers away.

14. Print preview allows a review of the document and the implementation of necessary changes before printing. True or False?


Answer:

True.

Explanation:

Print preview is a feature in some software applications that allows users to view their document in the format it will appear when printed. This feature allows users to review the document layout, font styles, images, and other elements before printing, which can help them identify any errors or formatting issues that need to be corrected. Additionally, print preview can also provide users with the option to make adjustments to the document, such as adjusting margins or page orientation, before sending it to the printer. Overall, print preview is useful for ensuring that the document is printed correctly and is ready for distribution or dissemination.
