13. Study the parts of the sewing machine and give each function of the following parts. Stop motion screw- Stitch regulator- Presser foot- Tension screw- Thread take- up lever-​


Stop motion screw - It is tightened to allow the needle move when sewing. It is loosened to make threading of bobbin possible.

Stitch regulator - is a tiny computer attached to a sewing machine.

Presser foot - is an attachment used with sewing machines to hold fabric flat as it is fed through the machine and stitched.

Tension screw - to control the gap between guide pulley.

Thread take - allow users to implement concurrent functions at the same time, which can either be dependent on each other for execution or independent. (I'm not sure)

up lever - It's the part of the machine that pulls the thread from the spool to feed it through the machine and lifts the thread back up out of the cloth after a stitch has been made.
